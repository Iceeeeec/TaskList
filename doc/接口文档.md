## 一、数据库设计

### ER 关系图

```
┌──────────┐     1:N     ┌──────────────┐
│ sys_user │────────────▶│ task_folder   │
└──────────┘             └──────────────┘
     │                          │
     │ 1:N                      │ 1:N
     ▼                          ▼
┌──────────┐             ┌──────────────┐     1:N     ┌───────────────┐
│ task_tag │             │  task_info   │────────────▶│ task_subtask   │
└──────────┘             └──────────────┘             └───────────────┘
     │                          │
     │         N:M              │
```

### 建表 SQL

```sql
CREATE TABLE `sys_user` (
  `id`          BIGINT       NOT NULL AUTO_INCREMENT COMMENT '主键',
  `username`    VARCHAR(50)  NOT NULL COMMENT '用户名',
  `password`    VARCHAR(255) NOT NULL COMMENT '密码',
  `nickname`    VARCHAR(50)  DEFAULT '' COMMENT '昵称',
  `avatar`      VARCHAR(255) DEFAULT '' COMMENT '头像URL',
  `create_time` DATETIME     DEFAULT CURRENT_TIMESTAMP COMMENT '创建时间',
  `update_time` DATETIME     DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP COMMENT '更新时间',
  `del_flag`    TINYINT(1)   DEFAULT 0 COMMENT '逻辑删除',
  PRIMARY KEY (`id`),
  UNIQUE KEY `uk_username` (`username`)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COMMENT='用户表';

CREATE TABLE `task_folder` (
  `id`          BIGINT       NOT NULL AUTO_INCREMENT COMMENT '主键',
  `user_id`     BIGINT       NOT NULL COMMENT '用户ID',
  `name`        VARCHAR(50)  NOT NULL COMMENT '文件夹名称',
  `color_index` INT          DEFAULT 0 COMMENT '颜色索引',
  `sort_order`  INT          DEFAULT 0 COMMENT '排序序号',
  `create_time` DATETIME     DEFAULT CURRENT_TIMESTAMP COMMENT '创建时间',
  `update_time` DATETIME     DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP COMMENT '更新时间',
  `del_flag`    TINYINT(1)   DEFAULT 0 COMMENT '逻辑删除',
  PRIMARY KEY (`id`),
  KEY `idx_user_id` (`user_id`)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COMMENT='清单文件夹表';

CREATE TABLE `task_tag` (
  `id`          BIGINT       NOT NULL AUTO_INCREMENT COMMENT '主键',
  `user_id`     BIGINT       NOT NULL COMMENT '用户ID',
  `name`        VARCHAR(20)  NOT NULL COMMENT '标签名称',
  `color_index` INT          DEFAULT 0 COMMENT '颜色索引',
  `create_time` DATETIME     DEFAULT CURRENT_TIMESTAMP COMMENT '创建时间',
  `del_flag`    TINYINT(1)   DEFAULT 0 COMMENT '逻辑删除',
  PRIMARY KEY (`id`),
  KEY `idx_user_id` (`user_id`)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COMMENT='标签表';

CREATE TABLE `task_info` (
  `id`             BIGINT        NOT NULL AUTO_INCREMENT COMMENT '主键',
  `user_id`        BIGINT        NOT NULL COMMENT '用户ID',
  `folder_id`      BIGINT        DEFAULT NULL COMMENT '所属文件夹ID',
  `title`          VARCHAR(200)  NOT NULL COMMENT '任务标题',
  `description`    VARCHAR(1000) DEFAULT '' COMMENT '任务描述',
  `priority`       VARCHAR(10)   DEFAULT 'mid' COMMENT '优先级 high/mid/low',
  `due_date`       DATETIME      DEFAULT NULL COMMENT '截止日期',
  `completed`      TINYINT(1)    DEFAULT 0 COMMENT '是否完成',
  `completed_time` DATETIME      DEFAULT NULL COMMENT '完成时间',
  `reminder`       TINYINT(1)    DEFAULT 1 COMMENT '是否提醒',
  `note`           TEXT          COMMENT '备注',
  `sort_order`     INT           DEFAULT 0 COMMENT '排序序号',
  `create_time`    DATETIME      DEFAULT CURRENT_TIMESTAMP COMMENT '创建时间',
  `update_time`    DATETIME      DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP COMMENT '更新时间',
  `del_flag`       TINYINT(1)    DEFAULT 0 COMMENT '逻辑删除',
  PRIMARY KEY (`id`),
  KEY `idx_user_id` (`user_id`),
  KEY `idx_folder_id` (`folder_id`),
  KEY `idx_due_date` (`due_date`),
  KEY `idx_completed` (`completed`)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COMMENT='任务主表';

CREATE TABLE `task_subtask` (
  `id`          BIGINT       NOT NULL AUTO_INCREMENT COMMENT '主键',
  `task_id`     BIGINT       NOT NULL COMMENT '所属任务ID',
  `text`        VARCHAR(200) NOT NULL COMMENT '子任务内容',
  `done`        TINYINT(1)   DEFAULT 0 COMMENT '是否完成',
  `sort_order`  INT          DEFAULT 0 COMMENT '排序序号',
  `create_time` DATETIME     DEFAULT CURRENT_TIMESTAMP COMMENT '创建时间',
  `del_flag`    TINYINT(1)   DEFAULT 0 COMMENT '逻辑删除',
  PRIMARY KEY (`id`),
  KEY `idx_task_id` (`task_id`)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COMMENT='子任务表';

CREATE TABLE `task_tag_rel` (
  `id`      BIGINT NOT NULL AUTO_INCREMENT COMMENT '主键',
  `task_id` BIGINT NOT NULL COMMENT '任务ID',
  `tag_id`  BIGINT NOT NULL COMMENT '标签ID',
  PRIMARY KEY (`id`),
  UNIQUE KEY `uk_task_tag` (`task_id`, `tag_id`),
  KEY `idx_tag_id` (`tag_id`)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COMMENT='任务标签关联表';
```

## 二、接口文档

> 统一返回格式：`{ code: 200, msg: "success", data: {...} }`  
> 所有接口除登录/注册外均需 Header：`Authorization: Bearer {token}`

### 1. 用户模块

#### 1.1 POST `/api/user/login` 用户登录

**请求：**

```json
{ "username": "admin", "password": "123456" }
```

**响应：**

```json
{
  "code": 200,
  "data": {
    "token": "eyJhbGciOi...",
    "userId": 1,
    "nickname": "Admin User",
    "avatar": ""
  }
}
```

### 4. 任务模块（核心）

#### 4.1 GET `/api/task/list` 查询任务列表

**请求参数（Query）：**

| 参数    | 类型   | 必填 | 说明                                      |
| ------- | ------ | ---- | ----------------------------------------- |
| filter  | String | 否   | `all`(默认) / `today` / `week` / 文件夹ID |
| keyword | String | 否   | 搜索关键词（匹配标题/描述）               |

**响应：**

```json
{
  "code": 200,
  "data": [
    {
      "id": 1,
      "title": "完成季度项目报告",
      "description": "需要包含Q3的所有财务数据分析...",
      "priority": "high",
      "dueDate": "2026-03-04 18:00:00",
      "completed": false,
      "folder": { "id": 1, "name": "工作", "colorIndex": 0 },
      "tags": [{ "id": 1, "name": "紧急", "colorIndex": 0 }]
    }
  ]
}
```

**SQL 示例：**

```sql
SELECT t.*, f.name AS folder_name, f.color_index AS folder_color
FROM task_info t
LEFT JOIN task_folder f ON t.folder_id = f.id
WHERE t.user_id = ? AND t.del_flag = 0
-- filter=today:  AND DATE(t.due_date) = CURDATE()
-- filter=week:   AND t.due_date BETWEEN NOW() AND DATE_ADD(CURDATE(), INTERVAL 7 DAY)
-- filter=文件夹ID: AND t.folder_id = ?
-- keyword:       AND (t.title LIKE '%?%' OR t.description LIKE '%?%')
ORDER BY t.completed ASC, FIELD(t.priority,'high','mid','low'), t.create_time DESC
```

**请求：**

```json
{
  "title": "新任务标题",
  "priority": "mid",
  "folderId": 1,
  "tagIds": [1, 3],
  "dueDate": "2026-03-10 18:00:00"
}
```

#### 4.4 PUT `/api/task/update` 修改任务

**请求：**（仅传需要修改的字段）

```json
{
  "id": 1,
  "title": "修改后的标题",
  "description": "新描述",
  "priority": "high",
  "folderId": 2,
  "dueDate": "2026-03-15 12:00:00",
  "note": "备注内容",
  "reminder": true
}
```

#### 4.5 PUT `/api/task/toggle/{id}` 切换完成状态

> 完成时自动记录 completed_time，取消完成时清除

#### 4.6 DELETE `/api/task/delete/{id}` 删除任务

> 级联：逻辑删除子任务 + 删除标签关联

#### 4.7 DELETE `/api/task/clearCompleted` 清除所有已完成任务

#### 4.8 PUT `/api/task/tags` 更新任务标签

**请求：**

```json
{ "taskId": 1, "tagIds": [1, 3] }
```

> 先删后插 task_tag_rel
